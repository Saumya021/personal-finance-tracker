# Personal Finance Tracker

A simple and user-friendly web application to help users manage their daily finances efficiently. Users can track income and expenses, categorize transactions, and monitor their spending through an interactive dashboard.

---

## Features

- User Authentication (Register & Login)
- Add Income and Expense Transactions
- Categorize Transactions
- Edit and Delete Transactions
- Dashboard with Spending Summary
- Filter Transactions by Category and Date
- Responsive and Clean UI
- Secure Backend APIs

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
