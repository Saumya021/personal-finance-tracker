# Personal Finance Tracker

A full-stack Personal Finance Tracker web application built using the MERN stack.

## Features

* User Authentication (Login/Register)
* Add Income & Expense Transactions
* Filter Transactions
* Dashboard with Balance Summary
* Responsive UI with Tailwind CSS
* Secure JWT Authentication

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication

## Project Setup

### Clone Repository

```bash
git clone https://github.com/Saumya021/personal-finance-tracker.git
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

### Frontend Setup

```bash
cd client
npm install
npm start
```

## Environment Variables

Create a `.env` file inside `server` folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
